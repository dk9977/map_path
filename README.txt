This program finds the optimal path through a park, based on season, terrain, and elevation.
Note: must have Python library PIL to run
Usage: python3 lab1.py terrain.png mpp.txt "path_filename.txt" "season" "output_filename.png"
"path_filename.txt" should be any of simple_path.txt, red.txt, white.txt, brown.txt, or your own file in the same format
"season" should be either summer, fall, winter, or spring
"output_filename.png" should be the name of the file that the final map will be saved to
